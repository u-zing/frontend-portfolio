프로젝트 개요

Musinsa Talk은 무신사 브랜드의 감성을 바탕으로 제작한 카카오톡 클론 웹앱입니다. HTML과 CSS만으로 UI를 구현하며, 레이아웃 구성, 반응형 스타일링, 컴포넌트화된 디자인 요소 등을 직접 제작했습니다. 본 프로젝트를 통해 퍼블리싱 역량과 디자인 감각을 동시에 발전시켰습니다.

주요 기능

  • Friends 화면: 친구 목록 및 프로필 표시
  • Find 화면: 무신사 상품이미지 자동슬라이드 애니메이션과 친구 찾기 UI 구현
  • 반응형 디자인: 모바일 최적화

기술 스택

프로젝트 개요

HTML5 Canvas API와 JavaScript를 활용하여 브라우저에서 직접 사용할 수 있는 웹 기반 그림 그리기 애플리케이션입니다. 데스크톱 그림판의 핵심 기능들을 웹 환경으로 구현하여 별도의 소프트웨어 설치 없이도 창작 활동이 가능합니다.

주요 기능

  • 그리기/채우기 모드
  • 색상 팔레트
  • 선 굵기 조절
  • 이미지 삽입 및 파일저장

Chrome의 Momentum 확장프로그램을 클론한 할 일 관리 및 배경화면 서비스입니다. Local Storage를 활용하여 사용자 데이터를 브라우저에 저장하고, 외부 API 연동을 통해 실시간 날씨 정보를 표시합니다.

  • 배경화면 변경 / 배경음악 on/off

프로젝트 개요

반응형 디자인이 적용된 개인 포트폴리오 웹사이트입니다. 라디오 버튼을 활용한 CSS 기반 탭 네비게이션과 모달을 통한 프로젝트 상세 설명 기능을 구현했습니다. 폴더와 파일을 열어보는 듯한 직관적인 UI/UX 디자인으로 사용자 친화적인 인터페이스를 제공합니다.

주요 기능

  • 라디오 버튼 기반 탭 네비게이션 (JavaScript 없이 구현)
  • 모든 디바이스 대응 반응형 레이아웃
  • 모달을 활용한 프로젝트 상세 설명
  • CSS 애니메이션 및 호버 효과
  • 직관적인 폴더/파일 시스템 UI

Growth & Goals

🌱 FROM CURIOSITY TO CODE

처음에는 단순히 내가 작성한 코드가 화면에 구현되는 모습이 신기해서 시작했지만, 지금은 더 나은 사용자 경험을 제공하기 위해 고민하는 과정에서 큰 매력을 느끼고 있습니다. HTML과 CSS의 기초부터 시작해 반응형 웹사이트를 직접 구현하며 레이아웃 설계와 접근성의 중요성을 배웠습니다. 현재는 JavaScript를 활용해 동적인 인터랙션을 구현하고, React와 TypeScript 학습을 통해 코드의 확장성과 유지보수성을 높이는 방법을 익히고 있습니다. 앞으로는 프론트엔드 전반의 성능 최적화와 협업 역량을 강화해, 사용자에게 편리한 서비스를 제공하는 개발자로 성장하는 것이 목표입니다.

🎯 LEARNING ROADMAP
